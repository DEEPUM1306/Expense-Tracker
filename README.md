# Expense Tracker

## Overview

Expense Tracker is a web-based application that helps users manage their personal finances by tracking income, expenses, savings, and transaction history through an interactive dashboard.

## Features

* User Login Authentication
* Dashboard Overview
* Add Income and Expense Transactions
* Transaction History Management
* Monthly Financial Reports
* Expense Distribution Charts
* Savings Tracking
* Dark Mode Support
* Responsive User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js

## Project Structure

* Dashboard Page
* Transactions Page
* Reports Page
* Login Page
* Interactive Charts and Reports

## Future Enhancements

* Database Integration
* Export Reports to PDF/Excel
* Budget Planning Features
* User Profile Management
